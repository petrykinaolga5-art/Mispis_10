# Университетская информационная система (Mispis_10)

## Описание проекта

Данный проект представляет собой реализацию информационной системы для управления научно-педагогическим составом университета. Система позволяет управлять сотрудниками, факультетами, институтами, курсами и исследовательскими проектами.

## Структура проекта

### Классы системы

- **Employee** - абстрактный базовый класс для всех сотрудников
  - **ResearchAssociate** - научный сотрудник (наследуется от Employee)
    - **Lecturer** - преподаватель (наследуется от ResearchAssociate)
  - **AdministrativeEmployee** - административный сотрудник (наследуется от Employee)

- **Faculty** - факультет университета
- **Institute** - институт университета
- **Course** - учебный курс
- **Project** - исследовательский проект
- **Participation** - связующий класс для связи между ResearchAssociate и Project (многие-ко-многим)

## UML Диаграммы

Проект включает следующие UML диаграммы (созданные в Visual Paradigm):

1. **Use Case Diagram** - диаграмма прецедентов использования системы
2. **Class Diagram** - диаграмма классов с отображением всех связей
3. **Sequence Diagram** - диаграммы последовательности для основных use cases

Все диаграммы согласованы между собой:
- Объекты на диаграммах последовательности соответствуют классам на диаграмме классов
- Сообщения между объектами соответствуют методам классов
- Поведенческие диаграммы соответствуют use cases

## Запуск проекта

```bash
# Компиляция
javac -encoding UTF-8 *.java

# Запуск
java -Dfile.encoding=UTF-8 Main
```

Или используйте скрипт:
```bash
./run.sh
```
